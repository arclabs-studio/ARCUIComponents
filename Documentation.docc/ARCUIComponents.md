# ``ARCUIComponents``

Premium UI components for iOS, following Apple's Human Interface Guidelines.

## Overview

ARCUIComponents provides beautifully designed, reusable UI components that feel native to Apple's ecosystem. Built with **Clean Architecture**, **Swift 6**, and the latest **SwiftUI** features.

### Design Philosophy

- **Apple First**: Every component follows Apple's Human Interface Guidelines
- **Performance**: Optimized for smooth 60-120Hz animations
- **Accessibility**: Full VoiceOver, Dynamic Type, and reduced motion support
- **Type Safety**: Leverages Swift's type system for compile-time safety
- **Clean Code**: SOLID principles and separation of concerns
